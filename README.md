# Beproductive
Beproductive là ứng dụng quản lý thói quen và công việc giúp người dùng nâng cao hiệu suất làm việc hàng ngày.
## Features
- Tạo, quản lý task cá nhân
- Theo dõi tiến độ công việc
- Gợi ý thói quen tốt
- Dark mode / Light mode
- Thông báo nhắc nhở
